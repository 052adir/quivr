# תוכנית השקה — 14 יום (Fast Launch)

מטרה: מהאפס למנויים משלמים ראשונים תוך שבועיים, בעלות נמוכה, דרך טלגרם/וואטסאפ + שותפים.

---

## צ'קליסט טכני לפני השקה (Go-Live)

- [ ] לפרוס את האפליקציה לשרת ציבורי (Railway/Render/Fly/VPS) עם **HTTPS**.
- [ ] להגדיר `ANTHROPIC_API_KEY` ב-`.env` (מפעיל את המורה־AI).
- [ ] להגדיר `TELEGRAM_BOT_TOKEN` (מפעיל דחיפת התראות).
- [ ] לבדוק את משפך הדף: `/` (נחיתה) → טופס ליד → `/app` (הרשמה) → חיבור DEMO.
- [ ] להחליף את עדויות הדמה ב-`landing/index.html` בעדויות אמת ברגע שיש.
- [ ] **חיוב:** לחבר תשלום (Stripe/Paddle) לפני שגובים — כרגע הניסיון חינם ואין חיוב.
- [ ] להצפין מפתחות API של משתמשים (KMS/Vault) לפני קהל אמיתי.
- [ ] להגדיר דומיין ולוודא שכל קישורי ה-CTA מצביעים אליו.

> הערה: המוצר היום הוא MVP — חיוב אמיתי וניהול מנויים הם הצעד הטכני הבא לפני גבייה.

---

## שלב 0 — הכנה (יומיים)

- לסיים פריסה + HTTPS + מפתחות.
- להכין 3 קישורי מעקב: `?src=telegram`, `?src=whatsapp`, `?src=instagram`.
- לכתוב רשימת 20 יעדי שותפים (`influencer-affiliate.md`).
- לצלם 1 וידאו הדגמה קצר של הדאשבורד + התראה.

## שלב 1 — Soft launch (ימים 1–4)

- להפיץ הודעה A/B (`whatsapp-telegram-copy.md`) ב-3–5 קבוצות שבהן אתה כבר חבר.
- לפנות ל-15–20 משפיענים/אדמינים עם ה-DM (`influencer-affiliate.md`).
- יעד: 50–100 הרשמות לניסיון, 3–5 שותפים שמסכימים.
- לאסוף עדות אמת אחת לפחות → להעלות לדף הנחיתה.

## שלב 2 — Influencer push (ימים 5–10)

- לתת גישה חינם לשותפים שסגרו; לספק להם בריף + קוד `?ref=`.
- לתזמן 3–5 פוסטים/וידאו של שותפים על פני כמה ימים.
- להפעיל drip של 3 הודעות לכל ליד שהשאיר טלפון.
- יעד: 300–500 הרשמות מצטברות.

## שלב 3 — המרה (ימים 11–14)

- הודעת drip #3 לכל מי שמתקרב לסוף הניסיון ("נשארו יומיים").
- לשפר את הדף לפי הנתונים (איפה נושרים? איזה src ממיר הכי טוב?).
- יעד: המנויים המשלמים הראשונים + 2–3 שותפים פעילים שמייצרים זרם קבוע.

---

## מדדים לעקוב אחריהם (KPIs)

| מדד | איפה רואים | יעד שבועיים |
|-----|-----------|--------------|
| לידים שנלכדו | `GET /api/leads` | 200+ |
| הרשמות לאפליקציה | טבלת `users` | 300+ |
| חיבורים פעילים | טבלת `connections` | 100+ |
| לפי source / ref | `source`, `ref_code` בלידים | לזהות ערוץ מנצח |
| המרה לניסיון → תשלום | (אחרי חיבור חיוב) | 5%+ |

---

## הלופ של "אינטנסיבי ומהיר"

1. הפץ הודעה/קישור בערוץ.
2. מדוד `source`/`ref` — איזה ערוץ ואיזה שותף ממירים.
3. הכפל את מה שעובד, הרוג את מה שלא.
4. כל יום — עדות/הוכחה חברתית חדשה לדף.

> זכור בכל מסר: **read-only, חינוכי בלבד, בלי הבטחות רווח.** זה מה ששומר על אמון ועל אתה מחוץ לצרות רגולטוריות.

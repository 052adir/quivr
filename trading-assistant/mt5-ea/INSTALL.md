# התקנת מנטור ב-MetaTrader 5 — מדריך ללקוח

בוט קריאה-בלבד שמתקינים פעם אחת ב-MT5. הוא מציג פאנל חי על הגרף ומתריע בזמן
אמת על טעויות — **לא סוחר, לא נוגע בכסף.** מתאים לסוחרי MT5 על מחשב.

## התקנה (5 דקות, פעם אחת)
1. **הורד את הבוט** מהאפליקציה: "חיבור והגדרות" → "הורד את הבוט (MentorGuard.ex5)".
2. ב-MT5: **קובץ → פתח תיקיית נתונים** (Open Data Folder).
3. היכנס לתיקייה `MQL5\Experts` והעתק לשם את `MentorGuard.ex5`.
4. חזור ל-MT5, פתח את ה-**Navigator** (Ctrl+N) → תחת "יועצים מומחים" יופיע **MentorGuard**.
   *(אם לא — קליק ימני → רענן.)*
5. ודא שכפתור **"מסחר אלגוריתמי"** בסרגל העליון דלוק (ירוק).
6. **גרור** את MentorGuard על הגרף → בחלון שנפתח לחץ **אישור**.

הפאנל יופיע בפינת הגרף עם החשבון, הפוזיציות, מצב הסטופ ואזהרות.

## התראות לטלגרם (מומלץ)
כדי לקבל את האזהרות גם לטלגרם (בכל מכשיר):
1. באפליקציה → "חיבור והגדרות" → "חבר טלגרם" → עקוב אחר ההוראות (לחיצה אחת).
2. בהגדרות ה-EA (לשונית "קלט"/Inputs), מלא:
   - `InpBackendUrl` = כתובת השרת + `/api/ea/event`
   - `InpToken` = הטוקן שלך מהאפליקציה
3. אשר WebRequest: **כלים → אפשרויות → יועצים מומחים** → סמן "אשר בקשות דפדפן עבור
   רשימת כתובות" → הוסף את כתובת השרת → אישור.

## עדכון הדשבורד באפליקציה
אם תמלא גם `InpBackendUrl` = כתובת השרת + `/api/mt5/trades`, הבוט ישלח את
היסטוריית העסקאות לאפליקציה והדשבורד + האבחנה יתעדכנו אוטומטית.

## בטיחות
הבוט **קריאה בלבד** — לעולם לא פותח, משנה או סוגר עסקאות. אפשר להריץ על דמו או אמיתי.
